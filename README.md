# refrigerator
## 👨‍🏫 프로젝트 소개
[2luli]
[우리집 AI 냉장고]

[기본기능]
- 회원 관리 : 관리자, 매니저, 사용자
- 게시판 : 공지사항, 나의 재료 공유 게시판
- 레시피 : 추천 레시피, 나만의 레시피
- 재료 : 일반 및 특가 상품 구매, 나만의 냉장고 재료 관리
- 결재 : 구매, 택배

[추가기능]
- 다이어트 맞춤 레시피 추천
- Like 레시피 추천 정렬
- 나만의 냉장고 재료 관리 기능
- 여분 재료 공유 게시판

## 🧑‍🤝‍🧑 개발자 소개
- **강수현** : 사용자, 매니저, 관리자, 소셜 로그인, 레시피, Like, 택배 및 결제 기능
- **조서현** : 나만의 냉장고, 다이어트 설정과 설정 상태에 따른 레시피 추천 기능
- **송기영** : 메인 페이지 기본 UI, 장바구니 재료 관리, 전반적인 게시판 기능
- **양혁진** : 회원정보 조회 및 변경, 회원 탈퇴, Like 레시피 연결 기능

## ⚙️ 기술 스택
- **Skill** : Spring Boot
- **Server** : Apache Tomcat v9.0
- **DataBase** : MySQL
- **NCP** : Naver Cloud Platform
- **아이디어 회의** : Slack, Zoom

## 📌 주요 기능
- 결제 및 배송
- 냉장고 재료, 다이어트 상태 기반 레시피 추천
- 나만의 냉장고 재료 관리
- 재료 공유
